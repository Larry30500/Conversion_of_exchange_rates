### 3. 第 1 個檢測事項：檢查被輸入的臺幣金額是某有誤，有三個檢測事項。
1. 被輸入的資料並非為整數或浮點數，則顯示輸入錯誤，並重新要求輸入。
2. 被輸入的金額小於零時，則顯示輸入錯誤，並要求重新輸入。
3. 使用者輸入 Q 或者 q，則退出程式。
* 輸入正確，則前往第 2 個檢測事項。

1. 使用者輸入僅限為整數 1 ~ 5 之間，否則顯示輸入錯誤。
2. 使用者輸入若非整數，則顯示輸入錯誤。
3. 輸入正確，則前往第 3 個檢測事項。
